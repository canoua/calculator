# Калькулятор

## Установка

1. git clone https://github.com/canoua/calculator
2. cd calculator
3. npm install
4. npm run dev

## Описание приложения

- Реализовано выполнение все основных мат. действий(+,-,\*,/);
- 'C' - обнуление строк;
- '%' - процент от числа;

## Навыки

- npm
- scss
- адаптив
- рендер дом дерева с помощью нативного js
- js(числа, строки, this)
